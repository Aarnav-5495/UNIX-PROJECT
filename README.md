# UNIX-PROJECT
This project generates a detailed report about system performance. It provides information on CPU load, memory usage, disk space, and uptime. You can view each report individually or see a full summary of the system health
# How to run this project:
Download or Clone the Repository
git clone https://github.com/<Aarnav-5495>/system-health-report.git
Navigate to the Project Folder
cd system-health-report
Give Execute Permission
chmod +x system_health_report.sh
Run the Script
./system_health_report.sh
# How to download this as file 
Go to your GitHub repository page.
Click the green “Code” button.
Select “Download ZIP”.
Extract the ZIP file to your desired folder.
Open a terminal in that folder and run:
chmod +x system_health_report.sh
./system_health_report.sh
